# Huishoudelijke Taken

Een eenvoudige webapplicatie om huishoudelijke taken bij te houden.

## Functies

- Taken toevoegen met een categorie (Keuken, Badkamer, Woonkamer, Tuin, Boodschappen, Algemeen)
- Taken afvinken als afgerond
- Taken verwijderen
- Filteren op openstaand / afgerond / alle taken
- Taken worden opgeslagen in de browser (localStorage)

## Gebruik

Open `index.html` in een webbrowser — geen installatie of server vereist.

## Technologie

- HTML5
- CSS3
- Vanilla JavaScript
